## Detailed description
Quizlet Exporter adds a sidebar to Quizlet that lets you export your flashcards in seconds.

✅ FEATURES
- Export to CSV — open in Excel, Google Sheets, or any spreadsheet app
- Export to JSON — developer-friendly structured data
- Export to Markdown — perfect for Obsidian, Notion, and docs
- Copy to Clipboard — paste anywhere instantly
- Smart filename — automatically uses the set title + today's date
- Dark / Light mode — follows your preference
- Configurable CSV separator (comma, semicolon, tab)
- Include/exclude header row option
- Works on standard Quizlet study sets

🔒 PRIVACY
• No data is collected or sent to any server
• All processing happens locally in your browser
• Only runs on quizlet.com

💡 HOW TO USE
1. Open any Quizlet study set
2. Click the Quizlet Exporter icon in your toolbar
3. Click the refresh button to scan terms
4. Click Download or Copy for your preferred format
